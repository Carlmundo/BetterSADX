Usage of config.ini
The main keyboard layout is defined in the section [Layout 1].
The [General] section contains the following options:
AlternativeLayouts enables two additional keyboard layouts which can be defined in sections [Layout 2] and [Layout 3] similar to the first layout.
DisableMouse completely disables any mouse-related input in the game.

Below is an example layout for config.ini. 
//INI file begins
[General]
AlternativeLayouts=False
DisableMouse=False

[Layout 1]
Analog Up=Up
Analog Down=Down
Analog Left=Left
Analog Right=Right
Button A=X
Button B=Z
Button X=S
Button Y=A
Button Z=None
Button Start=Enter
Trigger L=Q
Trigger R=W
D-Pad Up=I
D-Pad Down=M
D-Pad Left=J
D-Pad Right=K
Center Camera=E
//INI file ends
